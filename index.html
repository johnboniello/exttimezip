Installation Instructions

    On Windows 10

  1. Click the "Download" button below.
  2. Open the .zip folder

    3. Once inside the folder

  3. Look to the top of the window. You will see a pink rectangle that says "Compressed Folder Tools". Under this you will see "Extract". Click "Extract".
  4. Click "Extract All"
  5. Change the location of where you want this to be installed. It could be the desktop or anyplace you want it to go.
  6. Make sure "Show extracted files when complete is checked." Click "Extract"
  7. OPTIONAL: Right click the .exe file and create a shortcut. This will make it easier to access the program. Drag that shortcut to your desktop.
  8. If on Windows 10 a blue window may pop up the first time you run this program. Click *more info* and click *run anyway*.
  9. Click yes on all pop-ups that you trust the source of this program (I hope you do!)

On Windows 7

  • Click the "Download" button below.
  • Open your downloads folder, or wherever you told the file to download

    • Once inside the folder

  • Find the .zip file "Extended_time_calculator_1_41.zip"
  • Right click the file.
  • Click "Extract All"
  • Change the location of where you want this to be installed. It could be the desktop or anyplace you want it to go.
  • Make sure "Show extracted files when complete is checked." Click "Extract"
